Modo Soporte

Con este modo, el robot está programado para llevar objetos de un punto a otro. El robot escanea el entorno donde se encuentra y traza la ruta más optima hacia el punto objetivo. Es funcionalidad es muy util cuando se prentende transportar objetos de un punto a otro evitnado así dejar de lado otro tipo de tareas con mayor importancia.

Modo Exploración

En la modalidad de exploración, el robot está programado para investigar zonas inexploradas, en caso de detectar algo que se encuentre fuera de sus parametros de busqueda avisará a sus propietarios indicandoles la posición. También cuenta con un sensor de gas, el cual irá avisando si se detecta algún gas nocivo en el ambiente mientras explora la zona.

Modo Vigilancia

Este modo fue desarrollado expresamente para facilitar la vigilancia de zonas ya reconocidas. El robot trazara la ruta más óptima para cubrir el máximo terreno posible, y si encontrase algún intruso, el robot informaría de la posición del intruso y de ser necesarió el intruso sería atacado por el robot.
